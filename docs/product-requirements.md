# プロダクト要求定義書（Product Requirements Document）

## アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | 部員募集アプリ |
| ターゲットユーザー | 野球好きの少年・およびその保護者 |
| 解決したい課題 | 部員募集に手間をかけずに増やしたい。子どもの野球好きを応援したい保護者の不安も解決したい |
| 解決策 | チームの活動・雰囲気・費用・指導方針を外部に公開し、体験入団の申し込みをアプリで受け付ける |

## ユーザーの課題

### 野球好きの少年
- 入団前にチームの雰囲気がわからない
- 体験入団の申し込み方法がわからない

### 保護者
- 費用がどのくらいかかるかわからない
- 練習の頻度・場所がわからない
- 指導者の雰囲気・方針がわからない
- 送迎の負担がどのくらいかわからない

### チーム運営者（管理者）
- 体験申込者のスケジュール管理が大変
- 体験会の情報を申込者に一斉共有するのが手間

## 差別化要素

- チームの活動内容・雰囲気を写真や説明で紹介できる
- 試合結果・活動写真を外部に公開できる
- 体験入団の申し込みをアプリ内で受け付けられる
- 練習場所・練習日程を公開できる
- 費用（入会金・月会費・用具代など）をわかりやすく掲載できる
- 指導者のプロフィール・指導方針を公開できる
- 体験申込者の一覧をアプリで管理できる
- 体験会の日程・場所・集合情報を申込者に一斉共有できる

## 成功の定義

- チームの活動内容・雰囲気を外部の人が見られる
- 保護者が費用・練習場所・指導方針を事前に確認できる
- 体験入団の申し込みがアプリからできる
- 試合結果・活動写真を公開できる
- 体験申込者の一覧を管理できる
- 体験会の情報を申込者に一斉共有できる
- スマホでも見やすいデザイン

## 対象外（今回作らないもの）

- チーム内の管理機能（スケジュール・当番・配車など）
- プッシュ通知
- チャット機能
